# JarView

Jarview is a 3d photo organization visualization tool that helps you find and organize tasks.

- Tag your photos based on what is in the photos
- 3D visual world using touch designer
- Talking to an AI that can use tools to find and modify images via elevenlabs
- Using the mac photo libraries
